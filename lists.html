TYPES of LISTS

Ordered list-

it is the numbered list

example-

  1. HTML
  2. CSS
  3. JavaScript

Unordered List-

it is the bulleted list

example-

  • HTML
  • CSS
  • JavaScript

Description List

displays list of terms and their descriptions.contains dl , dt , dd

example-

HTML
HyperText Markup Language
CSS
Cascading Style Sheets

Nesting List

List inside a list

example-

  • frontend
    • HTML
    • CSS
    • JS
  • backend
    • node.js
    • express.js
  • Database

Mixed List

ordered and unordered list together

example-

  1. Register Account
    • Enter email
    • Verify otp
  2. Choose Plan
    • 1 Year
    • 6 Months
  3. Make Payment

Navigation List

each li contain anchor link to a section or page like navigation menu
